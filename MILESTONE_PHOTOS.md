## Important Notes

### Before Using Any Image:
1. **Verify the license** on the actual file page before downloading/using
2. **Wikimedia Commons**: Each file has its own license — click through to the file page and check the license section
3. **Unsplash/Pexels/Pixabay**: These platforms have blanket free-for-commercial-use licenses, but verify on the specific photo page
4. **NASA images**: US Government works are public domain — no restrictions on commercial use
5. **Attribution**: Even when not required, crediting photographers is good practice

### Recommended Priority Sources:
- **Best for guaranteed free commercial use**: Unsplash, Pexels, Pixabay, NASA
- **Best for Everest-specific content**: Wikimedia Commons (widest selection of trek-specific photos)
- **Best for trail map**: NASA Earth Observatory (public domain satellite imagery)

### Download Strategy:
For each milestone, visit the primary URL first. If the image doesn't match the needed vibe:
1. Try the alternative URLs
2. Browse the search pages linked above
3. For Wikimedia categories, browse the full category page for the best individual file
